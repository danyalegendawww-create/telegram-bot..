# Telegram Shop Bot

Бот для продажи привилегий и разбана на Minecraft сервере через Telegram Stars.

## 📋 Требования

- Python 3.11+
- Telegram Bot Token (получить у @BotFather)
- Telegram Stars (для тестирования платежей)
